## Цель проекта

## Данные

Архив объявлений о продаже квартир в Санкт-Петербурге и соседних населённых пунктах за несколько лет из сервиса Яндекс.Недвижимость

Требуемая память: 3.7+ MB
Строк: 23699 
Столбцов: 22
